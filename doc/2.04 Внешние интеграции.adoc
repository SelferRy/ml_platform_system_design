=== Внешние интеграции

[cols="2,2,2,2"]
|===
| Система | Протокол | Направление данных | Критичность

| SSO / Identity Provider
| OAuth 2.0 / OIDC
| ← (аутентификация пользователей)
| Критическая

| Объектное хранилище (S3/MinIO)
| S3 API (HTTPS)
| ↔ (артефакты моделей, inference-данные клиентов)
| Критическая

NOTE: S3 на диаграммах C4 показан как _внешняя система_ (не внутренний компонент), т.к. может быть как managed-сервис облака, так и BYOB (Bring Your Own Bucket) клиента. Единообразно на всех диаграммах.

| Стек monitoring (Prometheus)
| Prometheus remote write
| → (метрики)
| Высокая

| Алертинг (PagerDuty/Slack)
| Webhook / API
| → (алерты)
| Средняя

| CI/CD система (GitLab/Jenkins)
| REST API / Webhook
| ← (триггер развёртываний)
| Средняя

| Container Registry
| Docker Registry API
| ← (образы контейнеров)
| Критическая

| Kubernetes API
| K8s API
| → (развёртывание, масштабирование)
| Критическая
|===
