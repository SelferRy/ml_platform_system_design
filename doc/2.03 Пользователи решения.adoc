=== Пользователи решения

[cols="2,3,2,2"]
|===
| Роль | Действия | Контекст | Частота

| Data Scientist
| Загрузка моделей, просмотр метрик, запуск развёртываний, анализ предсказаний
| Внутренний
| Ежедневно

| ML Engineer
| Настройка конвейеров, настройка monitoring, управление инфраструктурой
| Внутренний
| Ежедневно

| Администратор платформы
| Управление квотами, контроль доступа, настройки инфраструктуры
| Внутренний
| Еженедельно

| Разработчик приложений
| Вызов API инференса, интеграция предсказаний в приложения
| Внутренний
| По запросу (автоматизировано)

| SRE / DevOps
| Monitoring здоровья платформы, обработка инцидентов, планирование ёмкости
| Внутренний
| Ежедневно

| **Бизнес-клиент (потребитель API)**
| Использование inference API в своих продуктах, получение предсказаний, оплата потребления
| Внешний
| Постоянно (автоматизировано)

| Менеджер продукта / Владелец бюджета
| Просмотр отчётов по использованию, управление бюджетами тенантов, согласование квот
| Внутренний
| Еженедельно
|===
